Dátum konvertáló
A program felhasználója megad egy dátumot (ÉÉÉÉ.HH.NN). Ezután a program ki ír adadtokat:

Hány nap telt el a beírt dátum óta: pl. 77
Hányadik nap az évben: pl. 5
Milyen napra esett: pl. Hétfő

Az eredményeket lelehet menteni egy szöveges fájlba tárolás céljából.
A program két modulból áll:
Egy modult amiben feldolgozzuk a beirt adatokat. És egy programot ami megvalósitja a felhasználói felületet és az eseménykezelést.
